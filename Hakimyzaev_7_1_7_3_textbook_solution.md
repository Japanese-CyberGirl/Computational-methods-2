# Домашняя работа: задачи 7.1-7.3

Решение написано по логике §7 учебника Хакимзянова и Черного:

1. привести схему к каноническому виду;
2. проверить устойчивость через ограниченность степеней оператора перехода;
3. для задач на сходимость проверить порядок аппроксимации;
4. применить теорему: аппроксимация + устойчивость дают сходимость того же порядка.

Обозначим длину отрезка

```math
l=b-a,
```

и равномерную сетку

```math
x_j=a+jh,\qquad hN=b-a=l.
```

## Что берем из учебника

### 1. Нормы

Для сеточного решения одной функции используем норму

```math
\|y_h\|_{U_h}=\max_{0\le j\le N}|y_j|.
```

Для правой части задачи Коши с одним начальным условием:

```math
\|f_h\|_{F_h}
=\max\left(|y_a|,\max_j|\varphi_j|\right).
```

Для двухшаговой схемы, где заданы \(y_0\) и \(y_1\), будем брать

```math
\|f_h\|_{F_h}
=\max\left(|y_0|,|y_1|,\max_j|\varphi_j|\right).
```

### 2. Устойчивость

По определению из §7 линейная разностная схема устойчива, если ее решение
существует единственно и выполняется оценка

```math
\|y_h\|_{U_h}\le C_y\|f_h\|_{F_h},
```

где \(C_y\) не зависит от шага \(h\).

### 3. Канонический вид

В учебнике схема приводится к виду

```math
y_{j+1}=R_hy_j+h\widetilde{\rho}_j.
```

Если степени оператора перехода равномерно ограничены:

```math
\|R_h^j\|\le C_4,\qquad j=1,\ldots,N,
```

то схема устойчива. Для скалярной схемы это означает просто

```math
|R_h^j|\le C_4.
```

В простом скалярном случае, если

```math
|\widetilde{\rho}_j|\le \|f_h\|_{F_h},
```

и

```math
|y_0|\le \|f_h\|_{F_h},
```

то из формулы решения следует учебниковая оценка

```math
\|y_h\|_{U_h}\le C_4(1+l)\|f_h\|_{F_h}.
```

### 4. Теорема о сходимости

Из §7: если разностная схема аппроксимирует дифференциальную задачу с порядком
\(p\) и устойчива, то решение разностной схемы сходится к точному решению с
порядком \(p\):

```math
\max_j|y_j-u(x_j)|\le Ch^p.
```

# Задача 7.1

Дана схема

```math
\frac{y_{j+1}-y_j}{h}+\alpha y_{j+1}=\varphi_j,
\qquad j=0,\ldots,N-1,
\qquad \alpha=\mathrm{const}>0,
```

```math
y_0=y_a.
```

Нужно доказать устойчивость и оценить сверху постоянную \(C_y\).

## Шаг 1. Приводим к каноническому виду

Умножим уравнение на \(h\):

```math
y_{j+1}-y_j+\alpha h y_{j+1}=h\varphi_j.
```

Соберем слагаемые с \(y_{j+1}\):

```math
(1+\alpha h)y_{j+1}=y_j+h\varphi_j.
```

Делим на \(1+\alpha h\):

```math
y_{j+1}
=\frac{1}{1+\alpha h}y_j
+h\frac{\varphi_j}{1+\alpha h}.
```

Это канонический вид

```math
y_{j+1}=R_hy_j+h\widetilde{\rho}_j,
```

где

```math
R_h=\frac{1}{1+\alpha h},
\qquad
\widetilde{\rho}_j=\frac{\varphi_j}{1+\alpha h}.
```

## Шаг 2. Проверяем ограниченность степеней \(R_h\)

Так как

```math
\alpha>0,\qquad h>0,
```

то

```math
1+\alpha h>1,
```

и поэтому

```math
0<R_h=\frac{1}{1+\alpha h}<1.
```

Следовательно,

```math
|R_h^j|\le 1,\qquad j=1,\ldots,N.
```

То есть в обозначениях учебника можно взять

```math
C_4=1.
```

## Шаг 3. Оцениваем правую часть

Поскольку

```math
\widetilde{\rho}_j=\frac{\varphi_j}{1+\alpha h},
```

а \(1+\alpha h>1\), имеем

```math
|\widetilde{\rho}_j|\le |\varphi_j|.
```

Значит,

```math
\max_j|\widetilde{\rho}_j|
\le \max_j|\varphi_j|
\le \|f_h\|_{F_h}.
```

Также

```math
|y_0|=|y_a|\le \|f_h\|_{F_h}.
```

## Шаг 4. Получаем оценку устойчивости

По учебниковой оценке для канонического вида

```math
\|y_h\|_{U_h}\le C_4(1+l)\|f_h\|_{F_h}.
```

Так как \(C_4=1\), получаем

```math
\|y_h\|_{U_h}\le (1+l)\|f_h\|_{F_h}.
```

Значит, схема устойчива.

## Ответ к 7.1

```math
\boxed{\text{схема устойчива, можно взять } C_y\le 1+l,\quad l=b-a.}
```

# Задача 7.2

Дана схема

```math
\frac{y_{j+1}-y_j}{h}
+\alpha\frac{y_j+y_{j+1}}{2}
=\varphi(x_{j+1/2}),
\qquad j=0,\ldots,N-1,
```

```math
y_0=y_a,
```

где

```math
x_{j+1/2}=x_j+\frac h2.
```

Она построена для задачи Коши

```math
y'(x)+\alpha y(x)=\varphi(x),
\qquad a\le x\le b,
\qquad \alpha=\mathrm{const}>0,
```

```math
y(a)=y_a.
```

Нужно исследовать схему на сходимость.

## Часть 1. Аппроксимация

Пусть \(u(x)\) - точное решение дифференциальной задачи. Нужно подставить
точное решение в схему и посмотреть на невязку.

Обозначим середину отрезка:

```math
x_m=x_{j+1/2}=x_j+\frac h2.
```

Тогда

```math
x_j=x_m-\frac h2,
\qquad
x_{j+1}=x_m+\frac h2.
```

Разложим \(u(x_{j+1})\) и \(u(x_j)\) в ряд Тейлора около точки \(x_m\):

```math
u(x_{j+1})
=u(x_m)+\frac h2u'(x_m)+\frac{h^2}{8}u''(x_m)
+\frac{h^3}{48}u'''(x_m)+\frac{h^4}{384}u^{(4)}(x_m)
+\frac{h^5}{3840}u^{(5)}(x_m)+O(h^6),
```

```math
u(x_j)
=u(x_m)-\frac h2u'(x_m)+\frac{h^2}{8}u''(x_m)
-\frac{h^3}{48}u'''(x_m)+\frac{h^4}{384}u^{(4)}(x_m)
-\frac{h^5}{3840}u^{(5)}(x_m)+O(h^6).
```

Вычтем второе из первого:

```math
u(x_{j+1})-u(x_j)
=hu'(x_m)+\frac{h^3}{24}u'''(x_m)
+\frac{h^5}{1920}u^{(5)}(x_m)+O(h^7).
```

Делим на \(h\):

```math
\frac{u(x_{j+1})-u(x_j)}{h}
=u'(x_m)+\frac{h^2}{24}u'''(x_m)+O(h^4).
```

Теперь сложим \(u(x_j)\) и \(u(x_{j+1})\):

```math
u(x_j)+u(x_{j+1})
=2u(x_m)+\frac{h^2}{4}u''(x_m)
+\frac{h^4}{192}u^{(4)}(x_m)+O(h^6).
```

Делим на \(2\):

```math
\frac{u(x_j)+u(x_{j+1})}{2}
=u(x_m)+\frac{h^2}{8}u''(x_m)+O(h^4).
```

Подставляем в левую часть схемы:

```math
\frac{u_{j+1}-u_j}{h}
+\alpha\frac{u_j+u_{j+1}}{2}
=u'(x_m)+\alpha u(x_m)
+h^2\left(\frac{1}{24}u'''(x_m)+\frac{\alpha}{8}u''(x_m)\right)
+O(h^4).
```

Но точное решение удовлетворяет дифференциальному уравнению:

```math
u'(x_m)+\alpha u(x_m)=\varphi(x_m).
```

Значит,

```math
\frac{u_{j+1}-u_j}{h}
+\alpha\frac{u_j+u_{j+1}}{2}
=\varphi(x_{j+1/2})+\psi_j,
```

где

```math
\psi_j
=h^2\left(\frac{1}{24}u'''(x_{j+1/2})
+\frac{\alpha}{8}u''(x_{j+1/2})\right)
+O(h^4).
```

Следовательно,

```math
\max_j|\psi_j|\le C_a h^2.
```

Начальное условие аппроксимируется точно:

```math
u(a)=y_a.
```

Итак, схема аппроксимирует задачу со вторым порядком:

```math
\boxed{p=2.}
```

## Часть 2. Устойчивость

Приведем схему к каноническому виду.

Исходная схема:

```math
\frac{y_{j+1}-y_j}{h}
+\alpha\frac{y_j+y_{j+1}}{2}
=\varphi(x_{j+1/2}).
```

Умножим на \(h\):

```math
y_{j+1}-y_j+\frac{\alpha h}{2}(y_j+y_{j+1})
=h\varphi(x_{j+1/2}).
```

Соберем \(y_{j+1}\) слева, \(y_j\) справа:

```math
\left(1+\frac{\alpha h}{2}\right)y_{j+1}
=
\left(1-\frac{\alpha h}{2}\right)y_j
+h\varphi(x_{j+1/2}).
```

Делим на \(1+\alpha h/2\):

```math
y_{j+1}
=R_hy_j+h\widetilde{\rho}_j,
```

где

```math
R_h=
\frac{1-\alpha h/2}{1+\alpha h/2},
\qquad
\widetilde{\rho}_j=
\frac{\varphi(x_{j+1/2})}{1+\alpha h/2}.
```

По определению устойчивости достаточно рассматривать малые \(h\). Возьмем

```math
h<h_0=\frac{2}{\alpha}.
```

Тогда

```math
0<1-\frac{\alpha h}{2}<1+\frac{\alpha h}{2},
```

поэтому

```math
0<R_h<1.
```

Значит,

```math
|R_h^j|\le 1,\qquad j=1,\ldots,N.
```

То есть можно взять

```math
C_4=1.
```

Также

```math
|\widetilde{\rho}_j|
=
\left|\frac{\varphi(x_{j+1/2})}{1+\alpha h/2}\right|
\le |\varphi(x_{j+1/2})|.
```

Следовательно,

```math
\max_j|\widetilde{\rho}_j|
\le \|f_h\|_{F_h}.
```

И

```math
|y_0|=|y_a|\le \|f_h\|_{F_h}.
```

По учебниковой оценке:

```math
\|y_h\|_{U_h}
\le (1+l)\|f_h\|_{F_h}.
```

Значит, схема устойчива.

## Часть 3. Сходимость

Мы доказали:

```math
\text{аппроксимация порядка }2,
```

и

```math
\text{устойчивость}.
```

По теореме из §7:

```math
\text{аппроксимация}+\text{устойчивость}\Rightarrow\text{сходимость}.
```

Поэтому

```math
\max_{0\le j\le N}|y_j-u(x_j)|\le Ch^2.
```

## Ответ к 7.2

```math
\boxed{\text{схема сходится со вторым порядком.}}
```

# Задача 7.3

Дана схема

```math
\frac{y_{j+1}-y_{j-1}}{2h}
+\alpha\frac{y_{j-1}+y_{j+1}}{2}
=\varphi_j,
\qquad j=1,\ldots,N-1,
```

```math
y_0=y_a,
```

```math
y_1=y_*=(1-\alpha h)y_a+h\varphi(a).
```

Она построена для той же задачи

```math
y'(x)+\alpha y(x)=\varphi(x),
\qquad y(a)=y_a,
\qquad \alpha=\mathrm{const}>0.
```

Нужно показать, что решение схемы сходится ко второму порядку.

## Часть 1. Аппроксимация внутреннего уравнения

Подставим точное решение \(u(x)\). Разложим около узла \(x_j\).

Для \(u(x_{j+1})\):

```math
u_{j+1}
=u_j+hu'_j+\frac{h^2}{2}u''_j+\frac{h^3}{6}u'''_j
+\frac{h^4}{24}u^{(4)}_j+\frac{h^5}{120}u^{(5)}_j+O(h^6).
```

Для \(u(x_{j-1})\):

```math
u_{j-1}
=u_j-hu'_j+\frac{h^2}{2}u''_j-\frac{h^3}{6}u'''_j
+\frac{h^4}{24}u^{(4)}_j-\frac{h^5}{120}u^{(5)}_j+O(h^6).
```

Вычтем:

```math
u_{j+1}-u_{j-1}
=2hu'_j+\frac{h^3}{3}u'''_j+\frac{h^5}{60}u^{(5)}_j+O(h^7).
```

Делим на \(2h\):

```math
\frac{u_{j+1}-u_{j-1}}{2h}
=u'_j+\frac{h^2}{6}u'''_j+\frac{h^4}{120}u^{(5)}_j+O(h^6).
```

Теперь сложим:

```math
u_{j-1}+u_{j+1}
=2u_j+h^2u''_j+\frac{h^4}{12}u^{(4)}_j+O(h^6).
```

Делим на \(2\):

```math
\frac{u_{j-1}+u_{j+1}}{2}
=u_j+\frac{h^2}{2}u''_j+\frac{h^4}{24}u^{(4)}_j+O(h^6).
```

Подставляем в схему:

```math
\frac{u_{j+1}-u_{j-1}}{2h}
+\alpha\frac{u_{j-1}+u_{j+1}}{2}
=u'_j+\alpha u_j
+h^2\left(\frac{1}{6}u'''_j+\frac{\alpha}{2}u''_j\right)
+O(h^4).
```

Так как точное решение удовлетворяет уравнению

```math
u'_j+\alpha u_j=\varphi_j,
```

получаем

```math
\frac{u_{j+1}-u_{j-1}}{2h}
+\alpha\frac{u_{j-1}+u_{j+1}}{2}
=\varphi_j+\psi_j,
```

где

```math
\psi_j
=h^2\left(\frac{1}{6}u'''_j+\frac{\alpha}{2}u''_j\right)
+O(h^4).
```

Значит,

```math
\max_j|\psi_j|\le C_a h^2.
```

Внутреннее уравнение аппроксимируется со вторым порядком.

## Часть 2. Аппроксимация второго начального значения

По условию

```math
y_1=y_*=(1-\alpha h)y_a+h\varphi(a).
```

Поскольку \(u(a)=y_a\), а из дифференциального уравнения

```math
u'(a)+\alpha u(a)=\varphi(a),
```

получаем

```math
u'(a)=\varphi(a)-\alpha u(a)=\varphi(a)-\alpha y_a.
```

Тогда

```math
y_*
=y_a+h(\varphi(a)-\alpha y_a)
=u(a)+hu'(a).
```

С другой стороны, по формуле Тейлора:

```math
u(a+h)=u(a)+hu'(a)+\frac{h^2}{2}u''(a)+O(h^3).
```

Следовательно,

```math
u(a+h)-y_*
=\frac{h^2}{2}u''(a)+O(h^3)
=O(h^2).
```

То есть второе начальное значение задано со вторым порядком.

## Часть 3. Канонический вид

Перепишем внутреннее уравнение:

```math
\frac{y_{j+1}-y_{j-1}}{2h}
+\alpha\frac{y_{j-1}+y_{j+1}}{2}
=\varphi_j.
```

Умножим на \(2h\):

```math
y_{j+1}-y_{j-1}+\alpha h(y_{j-1}+y_{j+1})
=2h\varphi_j.
```

Соберем \(y_{j+1}\) слева:

```math
(1+\alpha h)y_{j+1}
=(1-\alpha h)y_{j-1}+2h\varphi_j.
```

Делим на \(1+\alpha h\):

```math
y_{j+1}
=q_hy_{j-1}
+h\frac{2\varphi_j}{1+\alpha h},
```

где

```math
q_h=\frac{1-\alpha h}{1+\alpha h}.
```

Так как схема связывает \(y_{j-1}\), \(y_j\), \(y_{j+1}\), ее надо записать
в векторной канонической форме, как в примере 7.4 учебника.

Введем вектор

```math
\mathbf y_j=
\begin{pmatrix}
y_{j+1}\\
y_j
\end{pmatrix}.
```

Тогда

```math
\mathbf y_{j-1}=
\begin{pmatrix}
y_j\\
y_{j-1}
\end{pmatrix}.
```

Система принимает вид

```math
\mathbf y_j
=R_h\mathbf y_{j-1}+h\widetilde{\rho}_{j-1},
\qquad j=1,\ldots,N-1,
```

где

```math
R_h=
\begin{pmatrix}
0 & q_h\\
1 & 0
\end{pmatrix},
```

```math
\widetilde{\rho}_{j-1}
=
\begin{pmatrix}
\dfrac{2\varphi_j}{1+\alpha h}\\
0
\end{pmatrix},
```

а начальный вектор

```math
\mathbf y_0=
\begin{pmatrix}
y_1\\
y_0
\end{pmatrix}
=
\begin{pmatrix}
y_*\\
y_a
\end{pmatrix}.
```

## Часть 4. Устойчивость

Используем норму в двумерном пространстве \(Y\), как в учебнике:

```math
\left\|
\begin{pmatrix}
z_1\\
z_2
\end{pmatrix}
\right\|_Y
=\max(|z_1|,|z_2|).
```

Тогда норма матрицы равна максимуму сумм модулей элементов по строкам:

```math
\|R_h\|_Y
=
\max\left(|0|+|q_h|,\ |1|+|0|\right)
=\max(|q_h|,1).
```

Так как

```math
q_h=\frac{1-\alpha h}{1+\alpha h},
```

при достаточно малых \(h\) имеем

```math
|q_h|\le1.
```

Значит,

```math
\|R_h\|_Y\le1.
```

Следовательно,

```math
\|R_h^j\|_Y\le1,\qquad j=1,\ldots,N.
```

То есть в теореме 7.1 можно взять

```math
C_4=1.
```

Теперь проверим остальные оценки из теоремы.

Во-первых,

```math
\|y_h\|_{U_h}
=\max_j|y_j|
\le \max_j\|\mathbf y_j\|_Y.
```

Значит,

```math
C_1=1.
```

Во-вторых,

```math
\|\mathbf y_0\|_Y
=\max(|y_*|,|y_a|)
\le \|f_h\|_{F_h}.
```

Значит,

```math
C_2=1.
```

В-третьих,

```math
\|\widetilde{\rho}_{j-1}\|_Y
=
\left\|
\begin{pmatrix}
\dfrac{2\varphi_j}{1+\alpha h}\\
0
\end{pmatrix}
\right\|_Y
=\frac{2|\varphi_j|}{1+\alpha h}
\le 2|\varphi_j|.
```

Следовательно,

```math
\max_j\|\widetilde{\rho}_{j-1}\|_Y
\le 2\|f_h\|_{F_h}.
```

Значит,

```math
C_3=2.
```

По теореме 7.1:

```math
C_y\le C_1C_4(C_2+lC_3).
```

Подставляем

```math
C_1=1,\qquad C_4=1,\qquad C_2=1,\qquad C_3=2.
```

Получаем

```math
C_y\le 1+2l.
```

Значит, схема устойчива.

## Часть 5. Сходимость

Мы доказали:

1. внутреннее уравнение аппроксимируется со вторым порядком;
2. второе начальное значение задано со вторым порядком;
3. схема устойчива.

Значит, вся разностная задача аппроксимирует исходную задачу со вторым
порядком и устойчива.

По теореме о сходимости из §7:

```math
\max_{0\le j\le N}|y_j-u(x_j)|\le Ch^2.
```

## Ответ к 7.3

```math
\boxed{\text{решение схемы сходится ко второму порядку.}}
```
